# CodeAlpha_MatrixOperations
This project is a simple and beginner-friendly Matrix Operations program developed using the C programming language. The main purpose of this project is to perform common mathematical operations on matrices with the help of 2D arrays, functions, loops, and menu-driven programming.

In this project, the user can perform operations such as matrix addition, matrix multiplication, and transpose of a matrix. The program first asks the user to enter the size of the matrix and then takes matrix elements as input. After that, the selected operation is performed and the result is displayed in proper matrix format.

The project is designed in a modular way, where separate functions are used for reading the matrix, displaying the matrix, adding matrices, multiplying matrices, and finding the transpose. This makes the code clean, readable, and easy to understand.

This project is useful for students who want to strengthen their basic concepts of arrays and functions in C. It also helps in understanding how nested loops are used to access rows and columns of a matrix. Overall, this project gives practical experience in solving mathematical problems using programming.

Key Features : 
1. Matrix addition
2. Matrix multiplication
3. Matrix transpose
4. Menu-driven program
5. Use of 2D arrays
6. Use of functions for modular coding

Concepts Used :
1. C programming
2. 2D arrays
3. Functions
4. Nested loops
5. Conditional statements
6. Menu-driven logic
